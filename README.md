# stephencox1026.github.io — AI/ML Portfolio Gallery

**Live at https://stephencox1026.github.io**

A visual index of my AI/ML and data-science projects, grouped by skill area, each card linking to its GitHub repo (and live demo where available). Static site — HTML/CSS/JS, no build step.

## Files
- `index.html` — page shell
- `style.css` — styling
- `projects.js` — the project data (edit to add/adjust cards)
- `app.js` — renders and filters the cards
